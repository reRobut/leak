

# 根本故障根源
商铺饭店的动力配电箱附近，只有三相动力电线（三火一地）。老板买了一台 220V 的单相设备（商用烤箱，功率太大不能接普通插座）。老板自己接线，接的一根火线和一根地线。漏电保护器损坏。

# 场景
1. 这条街总闸（4个分闸，需要玩家一个一个试出来哪家漏电），墙带电，贴着纸条（注意：每天9点以后有漏电，左键触碰game over，右键触碰才能察看），测出哪家带电之后解锁其他两个场景
2. 饭馆总闸
3. 饭馆后厨：烤箱（上面一根粗黑线分出四根灰色的线，下面一红一蓝两根接了上面的两根灰线）

# 维修通关操作
1. 拉一根零线给烤箱接上
2. 漏保换新

# 商业电：网吧
1. 根本故障根源和现象
物业水泵火线接地漏电；水泵接水接通电时，变压器物业电闸a相b相c相可以测到电压，变压器网吧电闸a相电闸没有电压，变压器总闸a相也没有电压，网吧配电箱总闸a相也没有电压，网吧零线排带电，物业配电箱水泵轻微漏电（物业漏保轻微漏电不会跳闸），三相电a相漏电导致漏电流流入网吧零线排引发网吧配电箱漏保跳闸。这个时候网吧漏保可以合闸但是隔一会又会跳闸，并且触摸零线排，地下放置水泵的圆形管道，水泵外壳会触电。

2. 场景
  1. 变压器配电箱（三相总闸，网吧三相电闸，物业三相电闸）
  2. 网吧配电箱（三相总闸，零线排，网吧漏保）
  3. 物业配电箱（水泵漏保，水泵开关）
  4. 物业地下水泵火线勒在金属原型管道边缘，观察发现已经线勒扁，破皮部分勒穿了下面垫着的绝缘胶，火线直接与圆形管道接触。

3. 维修通关操作
用绝缘电缆皮包裹破损部位，扎带紧固固定
