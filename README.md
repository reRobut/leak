# 根本故障根源和现象
喷砂机（Sandblasting Cabinet）原本的绝缘手套破了，换了不绝缘的手套。导致喷砂过程中堆积的静电传导到工人身上。

# 场景
工人站在喷砂机前操作,身上带弱电。

# 维修通关操作
把手套更换成绝缘材料的手套。喷砂机接地。配备防静电鞋。
