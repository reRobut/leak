房屋入户其中一路火线绝缘外皮因绕线过紧破损，裸露火线铜芯直接搭接房屋金属钢结构、外墙钢筋、室外金属楼梯立柱等建筑金属构件，形成火线对地金属体持续性接地放电、漏电故障。室内总闸仅切断室内后端用电线路，无法断开前端入户进线电源，因此总闸断电后全屋金属结构依旧带电

# 场景
1. 室内：裸露铁质立柱（强电）、PVC包裹工字钢立柱（强电）、周边瓷砖地面（无电，发热）【关闭总闸对以上现象无效，直接摸一次扣一半血】
2. 总闸/配电箱
3. 楼体外线绕钢筋处（问题线冒烟）

# 维修通关操作
1. 剪线：剪开线皮，零火一根一根剪（故障现象消失）一次性剪两根当场去世
2. 给钢筋包绝缘层
3. 接线
