# 根本故障根源
1.	楼道公共配电箱内其他住户火线接头绝缘胶带包裹不严、铜线裸露，箱体金属柜门触碰裸露火线，配电箱柜体与整栋楼公共地线连通，整栋楼公共地线整体带上 220V 市电电压，地线彻底失效反成带电火线；
2.	一楼总进线配电箱存在火零接线左右反接（规范左零右火，实际左火右零），仅接线顺序错误，不影响常规电器供电，但会加剧线路电位紊乱；
3.	室内地线端子铰接接触不良，属于次要叠加问题，重新紧固铰接可消除室内端地线局部异常；
4.	拉断自家屋内漏保 / 空开无法切断公共地线带电，因为漏电源头在楼栋公共线路，不在自家室内回路。
衍生次要现象
一楼总闸完全拉下断电后，室内入户零火线仍测出 100V 左右感应电，属于长线缆未彻底断电、相邻线路耦合产生的感应电压，并非实质性漏电，


# 场景
1. 油烟机外壳
2. 室内配电箱漏保和总空开，从室外一根铰接地线接入室内配电箱的地线排
3. 楼道公共配电箱（两户新老绝缘黑胶带拼接线路6根线，2个地线2个火线2个零线）
4. 楼道总闸配电箱（包括总闸，2户漏保）

 
# 维修通关操作
1. 室内入户地线（室外接入室内配电箱插在地线排上的一根地线）拆开破皮绝缘黑胶带重新缠绕（没有维修公共配电箱只断开总闸配电箱的漏电本户漏保也会被电，）
2. 楼道公共配电箱，挑开碰在一起的无故障户火线和有故障户地线，拆开破皮绝缘黑胶带重新缠绕（没断开总闸配电箱或者没断开2户漏保都会被电）
